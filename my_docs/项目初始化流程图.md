# Operit AI 项目初始化流程详解

## 一、项目概述

**Operit AI** 是一款移动端 AI 智能助手应用，基于 **Android + Kotlin + Jetpack Compose** 构建，集成了 AI 对话、工具调用、本地模型推理（MNN/llama.cpp）、MCP/Skill 插件生态、Ubuntu 终端、工作流自动化等丰富功能。

### 项目技术栈

| 层级 | 技术 |
|------|------|
| 前端 UI | Jetpack Compose + Material 3 |
| 编程语言 | Kotlin (JVM 17) |
| 构建系统 | Gradle + Kotlin DSL |
| 数据库 | Room (SQLite) + ObjectBox |
| 本地模型 | MNN + llama.cpp (GGUF) |
| 终端环境 | Ubuntu 24 (Proot) |
| 动画/3D | DragonBones + MMD + Filament (glTF) |
| 插件系统 | MCP (Model Context Protocol) + Skill |
| 网络 | OkHttp + Retrofit |
| 依赖注入 | 手动单例模式 |

## 三、详细初始化流程

### 阶段一：操作系统层初始化

```
用户点击应用图标
    │
    ├──► Android 系统创建新进程
    │       • 分配内存空间
    │       • 加载 APK 中的 DEX 文件
    │       • 初始化 ART 虚拟机
    │
    ├──► 解析 AndroidManifest.xml
    │       • 检查声明的权限 (INTERNET, STORAGE, CAMERA 等)
    │       • 注册所有组件 (Activity, Service, Receiver, Provider)
    │       • 确定 Application 类: com.ai.assistance.operit.core.application.OperitApplication
    │
    └──► 创建 Application 实例
            • 调用构造函数
            • 准备进入 attachBaseContext()
```

**AndroidManifest.xml 关键配置：**

| 配置项 | 说明 |
|--------|------|
| `application android:name=".core.application.OperitApplication"` | 自定义 Application 入口 |
| `android:largeHeap="true"` | 申请大内存堆 |
| `android:enableOnBackInvokedCallback="true"` | 启用返回手势回调 |
| `android:usesCleartextTraffic="true"` | 允许明文 HTTP 通信 |
| 前台服务声明 | AI 服务、截屏服务、悬浮窗服务等 |
| 权限声明 | 50+ 项权限，涵盖存储、网络、系统、位置、电话等 |

### 阶段二：Application 层初始化 (OperitApplication)

#### 2.1 attachBaseContext() - 语言环境初始化

```
attachBaseContext(base: Context)
    │
    ├──► configureOpenMpEnvironment()
    │       • 设置 KMP_AFFINITY=disabled
    │       • 设置 OMP_PROC_BIND=false
    │       • 避免 OpenMP 多线程亲和性问题
    │
    └──► 应用语言设置
            • 读取用户保存的语言偏好
            • Android 13+: 使用 AppCompatDelegate.setApplicationLocales()
            • Android 12-: 使用 Configuration.updateConfiguration()
            • 创建本地化 Context
```

#### 2.2 onCreate() - 核心初始化序列

```kotlin
onCreate() {
    // 记录启动时间
    appStartupTimeMs = System.currentTimeMillis()
    instance = this
    
    // 步骤 1: OpenMP 环境配置
    configureOpenMpEnvironment()
    
    // 步骤 2: 应用图标状态管理
    AppIconManager.ensureComponentState(this)
    
    // 步骤 3: 崩溃恢复检测
    val isCrashReportRecoveryStartup = CrashRecoveryState.consumePendingCrashReportLaunch(this)
    if (!isCrashReportRecoveryStartup) {
        AppLogger.resetLogFile()  // 重置日志文件
    }
    
    // 步骤 4: WorkManager 初始化
    ensureWorkManagerInitialized()
    
    // 步骤 5: 启动时临时文件清理 (异步)
    launchCleanOnExitCleanup()
    
    // 步骤 6: Activity 生命周期管理器
    ActivityLifecycleManager.initialize(this)
    
    // 步骤 7: AI 消息管理器初始化
    AIMessageManager.initialize(this)
    
    // 步骤 8: 内置插件注册
    PluginRegistry.initializeBuiltins()
    
    // 步骤 9: 应用生命周期钩子分发
    AppLifecycleHookPluginRegistry.dispatchAsync(APPLICATION_CREATE)
    
    // 步骤 10: 启动 AI 前台服务 (如果需要)
    startGlobalAIForegroundServiceIfNeeded()
    
    // 步骤 11: 全局异常处理器
    Thread.setDefaultUncaughtExceptionHandler(GlobalExceptionHandler(this))
    
    // 步骤 12: JSON 序列化器初始化
    json = Json { serializersModule = SerializationSetup.module; ignoreUnknownKeys = true }
    
    // 步骤 13: 应用语言设置 (再次确认)
    initializeAppLanguage()
    
    // 步骤 14: 用户偏好管理器
    initUserPreferencesManager(applicationContext, defaultProfileName)
    
    // 步骤 15: 长期记忆自动保存轮询器
    memoryAutoSaveScheduler = MemoryAutoSaveScheduler(applicationContext, applicationScope)
    memoryAutoSaveScheduler.start()
    
    // 步骤 16: Android 权限偏好
    initAndroidPermissionPreferences(applicationContext)
    
    // 步骤 17: 角色卡管理器 (异步)
    CharacterCardManager.getInstance(applicationContext).initializeIfNeeded()
    
    // 步骤 18: 自定义表情初始化 (异步)
    CustomEmojiRepository.getInstance(applicationContext).initializeBuiltinEmojis()
    
    // 步骤 19: Shell 执行器上下文
    AndroidShellExecutor.setContext(applicationContext)
    
    // 步骤 20: Shower 虚拟屏环境
    ShowerEnvironment.shellRunner = OperitShowerShellRunner
    ShowerEnvironment.logSink = ShowerLogSink { ... }
    
    // 步骤 21: PDFBox 资源加载器
    PDFBoxResourceLoader.init(getApplicationContext())
    
    // 步骤 22: 语言工厂初始化
    LanguageFactory.init()
    
    // 步骤 23: 文本分词器预热 (异步)
    TextSegmenter.initialize(applicationContext)
    
    // 步骤 24: Waifu 消息处理器
    WaifuMessageProcessor.initialize(applicationContext)
    
    // 步骤 25: 数据库预加载 (异步)
    database.openHelper.writableDatabase
    
    // 步骤 26: 全局图片加载器 (Coil)
    globalImageLoader = ImageLoader.Builder(this)
        .okHttpClient(customOkHttpClient)
        .diskCache(50MB)
        .memoryCache(15% of available memory)
        .build()
    
    // 步骤 27: 图片池管理器
    ImagePoolManager.initialize(filesDir, preloadNow = false)
    
    // 步骤 28: 媒体池管理器
    MediaPoolManager.initialize(filesDir, preloadNow = false)
    
    // 步骤 29: Skill 仓库 ZIP 池
    SkillRepoZipPoolManager.initialize(filesDir)
    
    // 步骤 30: 后台重任务串行执行 (延迟 800ms)
    applicationScope.launch(Dispatchers.Default) {
        delay(800)
        
        // 30a: 图片池磁盘预加载
        ImagePoolManager.preloadFromDisk()
        
        // 30b: 媒体池磁盘预加载
        MediaPoolManager.preloadFromDisk()
        
        // 30c: AI 工具注册
        AIToolHandler.getInstance(this@OperitApplication).registerDefaultTools()
    }
    
    // 步骤 31: 工作流调度器初始化 (异步)
    WorkflowSchedulerInitializer.initialize(applicationContext)
    
    // 步骤 32: 数据库备份调度 (异步)
    RoomDatabaseBackupScheduler.ensureScheduled(applicationContext)
    
    // 步骤 33: 无障碍服务预绑定 (异步)
    UIHierarchyManager.bindToService(this@OperitApplication)
}
```

#### 3.2 onCreate() - UI 初始化流程

```kotlin
onCreate(savedInstanceState: Bundle?) {
    // 1. 基础设置
    super.onCreate(savedInstanceState)
    window.setBackgroundDrawableResource(android.R.color.black)
    
    // 2. Intent 处理
    handleIntent(intent)
    restoreRuntimeTaskViewVisibilityIfNeeded()
    
    // 3. 组件初始化
    initializeComponents()
    
    // 4. ANR 监控启动
    anrMonitor.start()
    
    // 5. 偏好监听器
    setupPreferencesListener()
    
    // 6. 显示配置 (高刷新率/高性能模式)
    configureDisplaySettings()
    
    // 7. 插件加载状态初始化
    pluginLoadingState.setAppContext(this)
    PluginLoadingStateRegistry.bind(pluginLoadingState, lifecycleScope)
    
    // 8. 设置 Compose 内容 (占位)
    setAppContent()
    
    // 9. GitHub OAuth 处理
    processPendingGitHubAuth()
    
    // 10. 更新管理器设置
    setupUpdateManager()
    
    // 11. 首次创建检查
    if (savedInstanceState == null) {
        performInitialChecks()  // ★ 关键流程
    } else {
        initialChecksDone = true
    }
    
    // 12. 返回键处理
    setupBackPressHandler()
}
```

#### 3.3 performInitialChecks() - 初始检查流程

```
performInitialChecks()
    │
    ├──► 1. 检查通知权限 (Android 13+)
    │       • 已授予 → 继续
    │       • 未授予 → 请求权限
    │       • 需要说明 → 显示 rationale
    │
    ├──► 2. 检查权限级别设置
    │       • 已设置 → 继续
    │       • 未设置 → showPermissionGuide = true
    │
    ├──► 3. 启动插件加载 (条件)
    │       • 协议已接受 AND 无需权限引导
    │       • startPluginLoading()
    │
    └──► 4. 标记检查完成
            initialChecksDone = true
            setAppContent()  // 重新渲染
```

### 阶段四：插件/MCP 加载阶段

#### 4.1 PluginLoadingState.initializeMCPServer()

```

    │
    ├──► 更新初始状态
    │       消息: "正在初始化..."
    │       进度: 5%
    │
    ├──► 获取 MCPLocalServer 实例
    │       MCPLocalServer.getInstance(context)
    │
    ├──► 获取 MCPRepository 实例
    │       MCPRepository(context)
    │
    ├──► 刷新插件列表
    │       mcpRepository.refreshPluginList()
    │
    ├──► 获取已安装插件集合
    │       installedPluginIds = mcpRepository.installedPluginIds.first()
    │
    ├──► 筛选需要启动的插件
    │       pluginsToStart = installedPlugins.filter { isEnabled }
    │
    ├──► 设置插件列表到 UI
    │       setPlugins(pluginsToStart)
    │
    ├──► 创建 MCPStarter
    │       mcpStarter = MCPStarter(context)
    │
    ├──► 创建进度监听器
```

## 六、初始化性能优化策略

| 策略 | 实现方式 | 效果 |
|------|----------|------|
| **异步初始化** | `applicationScope.launch(Dispatchers.IO)` | 避免阻塞主线程 |
| **延迟加载** | `by lazy { ... }` | 按需初始化，减少启动时间 |
| **后台串行** | `delay(800)` + `withContext(Dispatchers.IO)` | 错峰执行重任务 |
| **数据库预热** | `database.openHelper.writableDatabase` | 避免首次查询卡顿 |
| **缓存预加载** | `ImagePoolManager.preloadFromDisk()` | 提前加载磁盘缓存 |
| **文本分词预热** | `TextSegmenter.initialize()` | 避免首次搜索初始化 |
| **ANR 监控** | `AnrMonitor.start()` | 检测主线程阻塞 |
| **启动计时** | `AppLogger.d(TAG, "【启动计时】...")` | 精确测量各阶段耗时 |
